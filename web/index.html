How it works

Three steps from a public web page to a cited answer — the same knowledge graph powers this page's chat widget and the MCP tools an AI client can call.

1

Crawl

MAGPIE reads public pages across 37+ Montana .mt.gov agency sites — HTML, PDF, and DOCX.

2

Connect

It builds a per-agency knowledge graph, linking documents by topic, keyword, and meaning.

3

Answer

Questions are answered from that graph and cited straight back to the source agency page.

M·AG·P·I·E → Montana · AGency · Public · Information · Engine. Named for the black-billed magpie — a sharp, resourceful bird you'll find all over Montana.

💬 Try it. Tap Ask MAGPIE (or the chat button, bottom-right) and ask something like “How do I get a pesticide license?” or “How do I file a consumer complaint?” Answers come straight from public agency pages, with citations.

Add this chat to your own site

Copy the tag below and paste it just before the closing </body> tag of any agency page. No build step, no dependencies — the widget appears in the bottom-right corner.

<script src="https://YOUR-CHAT-HOST.example.com/widget.js"
        data-api="https://YOUR-CHAT-HOST.example.com"></script>

Replace YOUR-CHAT-HOST.example.com in both places with the address of your hosted answer service. data-api is where the widget sends questions; it must point at the live /chat API, not localhost. This is an example project — you host the MAGPIE chat API yourself.

    Guided agent samples

    Beyond open Q&A, the same knowledge graph can drive guided expert personas — agents with a goal, a staged flow, and a focused slice of the graph to draw on. Each persona is just a config file, so new ones drop in without code.

    Community Planning Guide

    Walks a local planner, economic developer, or prospecting company through building a planning document — or sizing up a Montana community for a project — one step at a time, grounded in the Department of Commerce Community Planning Platform with cited sources. It remembers where you are and assembles your plan as you go.

    1

    Orient & scope

    Building a plan, or evaluating a community? Which community, and which plan type?

    2

    Ground & scaffold

    Pulls the matching guidance and assembles a plan outline you watch take shape.

    3

    Fund & act

    Maps you to the right program (CDBG, Main Street, SLIPA…) and the next office to contact.

    Open as a standalone page

    Configured entirely in personas/community-planning-guide.yaml — scope, goals, stages, and voice in one file. The same persona is also exposed to AI clients as the community_planning_guide MCP tool.
