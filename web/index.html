Add this chat to your own site

Copy the tag below and paste it just before the closing </body> tag of any agency page. No build step, no dependencies — the widget appears in the bottom-right corner.

<script src="https://YOUR-CHAT-API.mt.gov/widget.js"
        data-api="https://YOUR-CHAT-API.mt.gov"></script>

Replace YOUR-CHAT-API.mt.gov in both places with the address of your hosted answer service. data-api is where the widget sends questions; it must point at the live /chat API, not localhost. Don't have a hosted API yet? Contact the Knowledge Network team to get one provisioned.
