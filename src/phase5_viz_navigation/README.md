# Phase 5: HTML Navigation Visualization

Interactive force-directed network visualization of HTML navigation structure with MIME-type based node sizing.

## Overview

Creates D3.js force-directed visualizations that show how web pages link together, with visual hierarchy based on document types:
- **Domain Root**: 50px (largest)
- **HTML Pages**: 20px (medium)
- **Documents**: 6-8px (smallest)

## Features

✅ **Force-Directed Layout**
- Natural clustering of related pages
- Adjustable link distances
- Interactive drag and zoom

✅ **MIME-Type Visual Hierarchy**
- Size indicates importance (domain > HTML > docs)
- Color-coded by type
- Clear visual distinction

✅ **Interactive Features**
- Click to highlight connected nodes
- Hover for tooltips
- Filter by type (all/HTML only/documents)
- Toggle labels
- Click nodes to open URLs

## Visualization Features

### Node Sizing
```python
Domain Root:    50px  # Homepage
HTML Pages:     20px  # Web pages
PDF Documents:   8px  # PDF files
DOCX Documents:  8px  # Word files
Other Docs:      6px  # Other types
```

### Color Scheme
- 🔴 Red (#e53935): Domain root
- 🔵 Blue (#1e88e5): HTML pages
- 🟠 Orange (#fb8c00): PDF documents
- 🟣 Purple (#8e24aa): DOCX documents
- 🟢 Green (#43a047): Other documents

### Link Distances
- HTML → HTML: 50px (very short)
- HTML → Document: 60px (short)
- Other: 70px (compact)

### Interactive Controls

| Control | Action |
|---------|--------|
| **Click Node** | Highlight connections (click again to open URL) |
| **Drag Node** | Reposition |
| **Mouse Wheel** | Zoom |
| **Drag Background** | Pan |
| **Hover Node** | Show tooltip |
| **Click Background** | Clear highlight |

### Buttons
- **Reset View**: Return to initial zoom/position
- **Toggle Labels**: Show/hide node names
- **Clear Highlight**: Remove selection
- **Show All**: Display all nodes
- **HTML Only**: Filter to HTML pages
- **Documents Only**: Filter to documents

## Connected Highlighting

When you click a node:
- Connected nodes get **orange border** (4px)
- Connected links turn **orange and thicker** (3px)
- Unconnected elements are **dimmed** (15% opacity)
- Labels automatically show for connected nodes

## Algorithm

### Hierarchy Construction

1. **Load Graph**: Read navigation JSON from Phase 3
2. **Build Relationships**: Create parent-child map from edges
3. **Find Roots**: Identify nodes with no parents
4. **Recursive Tree Building**: Depth-first construction of hierarchy
5. **Handle Multiple Roots**: Create synthetic root if needed

### Tree Layout

1. **D3 Tree Layout**: Calculate node positions
2. **Vertical Spacing**: Proportional to tree depth
3. **Collapsible Nodes**: Store children in `_children` when collapsed
4. **Smooth Transitions**: Animated expand/collapse

### Radial Layout

1. **D3 Tree Layout**: Calculate positions in polar coordinates
2. **Angular Distribution**: 360° spread across first level
3. **Radial Links**: Curved paths between nodes
4. **Rotated Labels**: Text follows radial direction

## Troubleshooting

**No navigation graphs found**
- Run Phase 3 first: `python -m src.phase3_navigation.cli --build-all`

**Empty visualization**
- Check that navigation graph has nodes and edges
- Verify JSON structure matches schema

**Layout issues**
- Try different layout type (tree vs radial)
- Radial works better for fewer, balanced trees
- Tree works better for deep, linear hierarchies
